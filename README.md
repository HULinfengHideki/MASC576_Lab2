# MASC576 Lab 2: Thermal Conductivity of SiC Thin Films
Simulation scripts, processed data, and report materials for MASC 576 Lab 2 on non-equilibrium MD analysis of thermal conductivity in SiC thin films.
This repository contains the simulation scripts, processed data, analysis tables, and report materials for MASC 576 Lab 2.
The project studies the thermal conductivity of SiC thin films using non-equilibrium molecular dynamics simulations at different temperatures, system sizes, and relaxation times.
